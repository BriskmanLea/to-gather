# This project is now under active development, updates coming soon

## ToGather is a personal productivity platform that brings together the essential tools for planning, self-development, and life management in one place.

Instead of using multiple apps for notes, tasks, habit tracking, finances, and personal goals, ToGather provides a customizable workspace where users can enable only the modules they need.

## ✨ Features

### Productivity

* Daily planner & To-do lists
* Notes with tags and full-text search
* Wishlist
* Monthly and yearly goals

### Self-development

* Habit tracker
* Gratitude journal
* Achievement journal
* Monthly review
* Yearly review
* Life balance wheel

### Personal management

* Finance tracker
* Books tracker
* Courses tracker
* Movies & TV tracker

### Customization

* User authentication
* Personal dashboard
* Enable/disable features
* Dark / Light theme
* Responsive design
